# java-cli-maven-failsafe-findbugs-jbehave-testng-hello-world

## Description
A POC for maven app using testng
and jbehave framework with findbugs
and failsafe plugin.

## Tech stack
- java
- maven
  - testng
  - findbugs
  - failsafe
  - jbehave

## Docker stack
- adoptopenjdk/maven-openjdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
